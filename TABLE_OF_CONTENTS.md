## ✅ 완성된 기능 목록

### 1. 사용자 기능
- [x] **신규 입장**
  - 입장 코드 입력
  - 닉네임 입력 (중복 체크)
  - 성별 선택 (남/여)
  - MBTI 입력 (4자리 대문자)
  - 팀 번호 선택 (선택사항, 1~6)
  - 등록 완료 화면

- [x] **재입장**
  - 입장 코드 + 닉네임 입력
  - 본인 팀 정보 확인
  - 팀 번호, 성별, MBTI 표시

- [x] **전체 팀 보기**
  - 6개 팀 카드 표시
  - 각 팀의 남/여 인원수
  - 총 인원수 표시

### 2. 관리자 기능
- [x] **인증**
  - 비밀번호 로그인 (qwer1234)
  - 세션 관리 (sessionStorage)
  - 로그아웃

- [x] **현황 통계**
  - 전체/코드별 참가자 수 조회
  - 남/여 인원 통계
  - 코드 선택 드롭다운

- [x] **팀별 현황**
  - 전체/코드별 팀 통계
  - 6개 팀 인원 분포
  - 남/여 인원수
  - 독립적인 코드 필터

- [x] **일일 코드 관리**
  - 코드 생성 (날짜 기반)
  - 활성화/비활성화 토글
  - 코드 삭제 (참가자 포함)
  - 코드별 참가자 상세보기

- [x] **코드별 상세보기**
  - 참가자 목록 (닉네임, 성별, MBTI)
  - 팀별 그룹화
  - 남/여/E/I 통계
  - 모달 팝업

- [x] **팀 설정**
  - 팀당 최대 인원 선택 (6/7/8명)
  - 설정 저장

- [x] **팀 랜덤 배정**
  - 고급 알고리즘 실행
  - 진행 상황 표시
  - 결과 통계 (배정 인원, E/I 통계)

### 3. 팀 배정 알고리즘
- [x] **인원 균등 분배**
  - 6개 팀에 최대한 균등하게
  - 최대 1명 차이

- [x] **성비 균형**
  - 남성이 2명 이상 많으면 차단
  - 여성이 많은 것은 허용
  - 예: 남3여3 ✅, 남4여2 ❌, 남2여4 ✅

- [x] **MBTI E/I 균형**
  - I 타입 과다 방지
  - E 타입으로 균형 맞춤
  - 팀별 E/I 통계

- [x] **이전 팀원 겹침 방지**
  - 같은 팀원 최대 2명까지
  - 3명 이상 겹침 차단
  - 이전 팀 정보 추적

- [x] **성별 교차 배정**
  - I 타입: 남→여→남→여
  - E 타입: 남→여→남→여
  - 균형 극대화

### 4. 데이터베이스
- [x] **daily_codes** (일일 코드)
